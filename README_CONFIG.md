# Конфигурация GRASP

Файл конфигурации — `config.php` (или `config.yaml`) в корне проекта, возвращает массив. Все ключи опциональны — используются значения по умолчанию.

## Минимальный config.php

```php
<?php
return [
    'database'  =>  [
        'host'  =>  __DIR__ . '/grasp.sqlite',
    ],
    'storage'   =>  [
        'path'  =>  __DIR__ . '/storage/',
    ],
    'logs'      =>  [
        'path'  =>  __DIR__ . '/logs/',
    ],
];
```

Если пути по умолчанию (`/opt/grasp/...`) не подходят — переопределите только их, остальное возьмётся из дефолтов.

---

## database

| Ключ | Тип | По умолчанию | Описание |
|------|-----|-------------|----------|
| `driver` | string | `sqlite` | Драйвер БД: `sqlite`, `postgresql`, `mysql`. |
| `host` | string | `/opt/grasp/db/grasp.sqlite` | Для SQLite — путь к файлу. Для PgSQL/MySQL — хост. |
| `port` | int | `5432` | Порт (PgSQL/MySQL). |
| `dbname` | string | `grasp` | Имя БД (PgSQL/MySQL). |
| `user` | string | `grasp` | Пользователь (PgSQL/MySQL). |
| `password` | string | `''` | Пароль (PgSQL/MySQL). |
| `charset` | string | `utf8mb4` | Кодировка соединения (MySQL). |

## cron

Настройки планировщика (`cron.php`).

| Ключ | Тип | По умолчанию | Описание |
|------|-----|-------------|----------|
| `lock_file` | string | `/tmp/grasp_cron.lock` | Путь к lock-файлу, предотвращающему параллельный запуск cron. |
| `lock_timeout` | int | `300` | Время в секундах, после которого lock-файл считается устаревшим. |
| `lock_check_pid` | bool | `true` | Если `true` — при превышении `lock_timeout` проверяется, жив ли процесс-владелец lock-файла (через `/proc/{pid}`). Предотвращает снятие lock-а, если git clone/fetch всё ещё выполняется. Отключите (`false`), если нет прав на чтение `/proc`. |
| `max_per_run` | int | `3` | Максимальное количество репозиториев, обрабатываемых за один запуск cron. |
| `retry_delay` | int | `300` | Задержка в секундах перед повтором после неудачного клонирования/обновления. |

## features

| Ключ | Тип | По умолчанию | Описание |
|------|-----|-------------|----------|
| `deferred_delete` | bool | `false` | Если `true` — репозитории помечаются как `deferred_delete` вместо мгновенного удаления с диска. |
| `allow_server_info` | bool | `false` | Если `true` — в интерфейсе появляется кнопка «Информация о сервере» (использование памяти, диска). |
| `show_detailed_logs` | bool | `false` | Если `true` — на вкладке «События» показываются все события (включая смены состояний) и столбец с типом события. Если `false` — только завершённые операции, ошибки и ручные триггеры. |

## default_update_interval

| Тип | По умолчанию | Описание |
|-----|-------------|----------|
| string | `7d` | Интервал обновления по умолчанию для новых репозиториев. Формат: число + суффикс (`m` — минуты, `h` — часы, `d` — дни). Примеры: `30m`, `2h`, `7d`. |

## access

IP-адреса и подсети для доступа к API и веб-интерфейсу.

| Ключ | Тип | По умолчанию | Описание |
|------|-----|-------------|----------|
| `admin_ips` | array | `['127.0.0.1', '192.168.111.1/24', '::1']` | Список IP/подсетей с полным доступом (администрирование). |
| `view_ips` | array | `['192.168.111.1/24', '0.0.0.0/0', '::/0']` | Список IP/подсетей с доступом только на чтение. `0.0.0.0/0` и `::/0` — открытый доступ. |

## git_http_backend

Настройка Git HTTP Backend (см. `README_GIT_BACKEND.md` — установка и настройка nginx).

| Ключ | Тип | По умолчанию | Описание |
|------|-----|-------------|----------|
| `enabled` | bool | `false` | Включить отображение clone-URL в интерфейсе. |
| `info_ref_auto_update` | bool | `false` | Автоматически запускать `git update-server-info` после clone/update (обновляет `info/refs` для git-http-backend). |
| `base_url` | string | `'http://grasp.local/git'` | Базовый URL, показываемый в интерфейсе для клонирования. Должен совпадать с location в nginx. Может быть относительным (`/git`) или абсолютным (`https://example.com/git`). |

## docker

При запуске в Docker используйте `docker/config.docker.php` (лежит рядом с `docker-compose.yml`). Пути в нём указывают на `/var/opt/grasp/...` — именно туда пробрасываются тома.
